■ 이슈 키: COMMONR-380
■ 제목: [BSX지원기능][BDP-954]Authentication - Door Control Admin 
============================================================

■ 테스트 스텝

[Step 1]
스텝 (Step)
1. Device> 관리자 설정

데이터 (Data)
1. Master Password 설정
2. 전체 관리자 설정
3. 장치 설정 관리자 설정
4. 사용자 관리자 설정

예상 결과 (Expected Result)
1. 사용자가 선택한 관리자로 설정되어야 한다.
> 관리자에 대해 총 10명으로 제한(Old Version)
  ex) 전체관리자 10명을 만들경우 다른 관리자에 대해 설정할 수 없음.
> 관리자에 대해 총 1000명으로 제한(New Version)
> Master Password는 1명 Only
------------------------------------------------------------
[Step 2]
스텝 (Step)
[전체 관리자 미설정 상태]
1. 장치 내에서 사용자에 사용자관리자 & 설정 관리자 설정
2. BioStar X에서 장치>고급에서 사용자관리자 & 설정 관리자 설정
3. 설정한 관리자 및 일반 사용자로 출입문 제어 진입 시도

데이터 (Data)
1. CE 인증 장치
2. CE 미인증 장치

예상 결과 (Expected Result)
1. 전체 관리자가 없는 상태에서 장치 설정 관리자로 출입문 제어 진입시도시 바로 진입이 가능해야 한다.
2. 사용자관리자&일반 사용자의 경우 해당 메뉴에 진입할 수 없으므로 관리자 인증에 대한 오류 메세지를 표시한다.
3. CE 미인증 장치 BioStar에서 전체관리자 미 설정 되어 있는 상태에서 사용자&설정 관리자 설정 시 BioStar 오류 메시지 출력
------------------------------------------------------------
[Step 3]
스텝 (Step)
1. Device> 전체관리자 설정
2. 사용자 관리자 & 설정관리자 설정
3. 설정한 관리자로 출입문 제어 진입 시도

데이터 (Data)
N/A

예상 결과 (Expected Result)
1. 인증모드와 상관없이 지문(얼굴)/카드(QR)/ID+PIN 인증으로만 동작되어야 한다.
> Door Control 진입 로그가 남는다(확인 필요)
------------------------------------------------------------
[Step 4]
스텝 (Step)
1. Device> 전체관리자 설정
2. 일반 사용자 설정
3. 일반 사용자로 출입문 제어 진입 시도

데이터 (Data)
N/A

예상 결과 (Expected Result)
1. 일반 사용자로 진입시도시 "권한 없음"으로 진입이 실패 되어야 한다.
------------------------------------------------------------
[Step 5]
스텝 (Step)
1. Device> 전체관리자 설정
2. 전체 관리자로 출입문 제어 진입 시도
3. 인증결과 출력유무 확인

데이터 (Data)
출입문제어시도 인증
=> Card(QR)/ID+PIN/Face/Finger

예상 결과 (Expected Result)
1. 출입문 제어메뉴에 대해 표시가 되어야 한다.
2. 출입문 제어 Menu 진입 로그를 확인한다.(확인 필요)
> Card(QR)/ID+PIN/Face/Finger
3. 관리자로 출입문제어시 인증결과를 출력하지 않는다.
------------------------------------------------------------
[Step 6]
스텝 (Step)

------------------------------------------------------------
[Step 9]
스텝 (Step)
1. 관리자 설정/미설정
2. 메뉴 진입 상태 > 인증 
3. 메뉴 진입 시도와 동시에 인증 시도

데이터 (Data)
N/A

예상 결과 (Expected Result)
1. 설정된 권한으로 출입문제어가 되어야 한다.
> 메뉴 진입 후 인증 동작이 안되는 지 확인
2. 관리자 미설정 상태에서 인증 시 메뉴 진입 시도로 인증 동작 안되는 지 확인한다.
> 타이밍 상 발생할 수도 있으나, 대부분 인증 동작이 안되는 지 확인
------------------------------------------------------------
