■ 이슈 키: COMMONR-207
■ 제목: Full Log 상태에서 서버와 연결시 무한반복으로 로그 올리는 문제
============================================================

■ 테스트 스텝

[Step 1]
스텝 (Step)
1. 새로 설치한 서버에 장치 추가
2. Full Log가 서버로 업로드됨을 확인
3. 서버종료
4. 장치에 여러번의 인증시도(로그가 쌓이게하는 목적)
5. 서버 재실행
6. 로그 업로드 여부 확인

데이터 (Data)
로그 업데이트: 자동
- BioStar2 Settings > Server > General에서 로그 업로드 자동으로 설정

예상 결과 (Expected Result)
1. 장치로그가 서버로 모두 업로드 된다.
2. 서버와 연결이 끊긴 이후에 발생한 로그에 대해서만 서버로 올린다.
------------------------------------------------------------
[Step 2]
스텝 (Step)
1. 새로 설치한 서버에 장치 추가
2. 수동으로 장치의 로그 업로드
3. Full Log가 서버로 업로드됨을 확인
4. 장치에 여러번의 인증시도(로그가 쌓이게하는 목적)
5. 수동으로 장치의 로그 재업로드
6. 로그 업로드 여부 확인

데이터 (Data)
로그 업데이트: 수동
- BioStar2 Settings > Server > General에서 로그 업로드 Manual로 설정

예상 결과 (Expected Result)
1. 장치로그가 서버로 모두 업로드 된다.
2. 수동으로 로그 업로드 후에 발생한 로그에 대해서만 서버로 올린다.
------------------------------------------------------------
